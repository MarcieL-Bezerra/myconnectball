# Connect Ball Challenge

Um jogo casual onde o objetivo é conectar pares de bolas coloridas sem cruzar os caminhos, com visual moderno e responsivo.

## Como jogar
- Clique em uma bola colorida para iniciar o traço.
- Arraste o mouse (ou toque) para desenhar o caminho até a outra bola da mesma cor.
- Os caminhos não podem se cruzar!
- Cada par de bolas representa uma vida (❤️). Se cruzar caminhos, perde uma vida e o traço some após 2 segundos.
- Ao perder todas as vidas, aparece "Game Over" e o jogo reinicia.
- Avance de fase conectando todos os pares corretamente.

## Funcionalidades
- Visual de jogo com interface responsiva.
- Exibição de vidas com corações.
- Reinício automático ao perder todas as vidas.
- Suporte a mouse e touch.

## Estrutura
- `index.html`: Interface principal do jogo.
- `css/styles.css`: Estilos e responsividade.
- `js/script.js`: Lógica do jogo.

## Instalação e uso
1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
